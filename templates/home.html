Gemini PDF Chatbot

Project Description

This project is a personal endeavor aimed at advancing machine learning skills, particularly with Large Language Models (LLMs). The chatbot utilizes the Retrieval-Augmented Generation (RAG) technique, Gemini API and langchain to enable seamless interaction with documents. Users may provide their own PDF documents and ask questions to the chatbot based its contents.

Tech Stack

  • Python (Flask): used for developing the webapp and chat model
  • Gemini API: used as the LLM agent for chatbot
  • Langchain: Model development tools
  • ChromaDB: Vector database

System Architecture

The model utilizes the RAG technique where when the user provides a document, the document is split into chunks which are then converted into vector embeddings using the Google's Generative AI embeddings from Langchain. these vector embeddings are then stores in a ChromaDB Vector database. when the user asks a question, the vector DB searches for the most relevant chunks of texts to the users question. these chunks are then sent to the LLM along with the user's question so that the LLM can answer based on the retrieved context. the diagram below demonstrates how this process works.
